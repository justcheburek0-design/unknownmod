# Unknown Mod

Серверный Fabric-мод для режима социальной дедукции. Он скрывает личности игроков, умеет переводить их в `ghost`-состояние и периодически запускает `reveal`, когда личность одного игрока становится видна всем.

## Возможности

- Подменяет отображаемое имя игроков на анонимное.
- Может подменять скин анонимного имени через SkinRestorer.
- Поддерживает ручной перевод игрока в `ghost` и обратное восстановление.
- Запускает автоматическое и ручное `reveal` с таймером, предупреждением и отменой.
- Отправляет кастомные сообщения о входе, выходе, смерти и раскрытии личности.
- Имеет режим `debug` для диагностики и более прозрачных логов/сообщений для операторов.

## Требования

- Minecraft `1.21.11`
- Java `21`
- Fabric Loader `0.16.5+`
- Fabric API
- Polymer Core
- SkinRestorer

Часть библиотек уже включена в сам jar, но Fabric, Polymer Core и SkinRestorer должны быть доступны на сервере.

## Установка

1. Соберите мод командой `./gradlew build`.
2. Возьмите jar из `build/libs/`.
3. Положите jar в папку `mods` на сервере Fabric.
4. Убедитесь, что рядом установлены Fabric API, Polymer Core и SkinRestorer.
5. Запустите сервер один раз, чтобы создался `config/unknown.yml`.

## Конфигурация

Файл конфигурации создаётся автоматически в `config/unknown.yml`.

Основные разделы:

- `anonymous` - имя и скин, которые будут показываться вместо реального профиля.
- `messages` - шаблоны сообщений о входе, выходе, убийстве и раскрытии личности.
- `revelation` - параметры таймера, длительности и порога игроков.
- `debug` - включает или выключает режим отладки.

Пример структуры:

```yaml
anonymous:
  name: JustPlayer
  skin:
    type: texture # texture | nickname
    texture: ""
    signature: ""
    nickname: ""

revelation:
  enabled: true
  interval-hours: 2
  duration-minutes: 15
  min-players: 3
  warning-minutes: 5

debug:
  enabled: false
```

Поддерживаемые плейсхолдеры в сообщениях:

- `%player%`
- `%killer%`
- `%minutes%`
- `%hours%`
- `%n%`
- `%time%`

Важно: `interval-hours` и `duration-minutes` фактически не опускаются ниже `1`, а `warning-minutes` может быть `0`, чтобы отключить предупреждение.

## Команды

Все команды требуют прав оператора или аналогичных прав `gamemaster`.

| Команда | Что делает |
| --- | --- |
| `/unknown reload` | Перечитывает `config/unknown.yml` и применяет изменения без рестарта. |
| `/unknown debug` | Переключает режим отладки. |
| `/unknown nickname <value>` | Меняет анонимное имя игроков. |
| `/unknown skin <value>` | Если указать `texture`, применяет сохранённый `texture/signature`. Иначе пытается найти скин по никнейму. |
| `/unknown ghost <player>` | Переключает игрока в `ghost` или возвращает его обратно. |
| `/unknown reveal player` | Запускает `reveal` на случайном подходящем игроке. |
| `/unknown reveal player <player>` | Запускает `reveal` на конкретном игроке. |
| `/unknown reveal cancel` | Отменяет активный `reveal`. |
| `/unknown reveal interval <hours>` | Меняет интервал между `reveal`-событиями. |
| `/unknown reveal duration <minutes>` | Меняет длительность `reveal`. |
| `/unknown reveal minplayers <n>` | Меняет минимальный порог игроков для запуска `reveal`. |
| `/unknown reveal status` | Показывает текущий статус таймера или активного `reveal`. |

Примечание по `minplayers`: `reveal` стартует только когда онлайн игроков больше, чем указанное значение. Например, при `minplayers = 3` нужно минимум `4` игрока онлайн.
